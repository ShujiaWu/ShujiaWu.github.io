    Redis 数据备份与恢复

    数据备份

    SAVE 命令用于数据备份

    BGSAVE 命令用于后台执行数据备份

    以上两个命令在 redis 安装目录中创建 dump.rdb 文件

    数据恢复

    将备份文件 (dump.rdb) 移动到 redis 安装目录并启动服务即可
