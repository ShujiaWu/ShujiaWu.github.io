    • 初始化npm、以及package说明

    【1】npm安装
      楼主使用的windows，所以直接安装NodeJS，NodeJS中有附带npm。直接到官网下载后一直无脑下一步(o゜▽゜)o☆

    【2】使用npm生成package.json文件

      然后根据提示填写信息。

    【3】package.json文件说明

    字段 说明
    name npm包的名称
    version 版本号
    description 描述
    main 入口，指向程序的主要项目。就是说，如果你包的名字叫 express，然后用户安装它，然后require(“express”)。
    dependencies 依赖项
    devDependencies 开发依赖项
    scripts 脚本
    keywords 关键字
    author 作者
    license 协议

    其中description、devDependencies为工程的依赖包，devDependencies会下载文档和测试用例。

    【4】依赖项可以手动输入，然后使用 npm install 下载依赖项

    【5】使用 npm install <包名> --save 下载依赖包并添加到description中。

    【6】使用 npm install <包名> --save-dev 下载依赖包并添加到devDependencies中。
