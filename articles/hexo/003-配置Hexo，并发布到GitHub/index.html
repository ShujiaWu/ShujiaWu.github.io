    • 配置Hexo，并发布到GitHub

    【1】建立与你用户名相对应的仓库，这是一个固定写法，比如我的用户名为 ShujiaWu ，那么我的仓库名为： ShujiaWu.github.io

    【2】建立完成后，可以使用automatic page generator生成默认的页面，查看效果。

    【3】建立关联，配置Hexo配置文件，配置发布的目标

        1、进入你的blog文件夹，找到_config.yml文件

        2、定位到文档的最底部，找到 deploy 配置项

        3、修改如下：

    【repository】仓库的地址
    【branch】分支

        4、安装hexo gitdeployer

    【4】执行发布命令

    【5】发布成功后，访问 http://[用户名]..github.io/ 即可，比如我的是 http://shujiawu.github.io/
